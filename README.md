# jiadongpo.github.io

安装服务：
npm install hexo-cli -g
npm install hexo-server —save 

初始化项目
hexo init HexoJiadongpo
cd HexoJiadongpo

下载主题：
 git clone https://github.com/theme-next/hexo-theme-next themes/next

npm install
hexo server
